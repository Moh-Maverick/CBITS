Secure File Sharing System

The Secure File Sharing System enables users to upload, share, and download files in a secure and decentralized manner. This project leverages blockchain technology to ensure file integrity and privacy, while providing robust authentication for users.

Key features include encryption of uploaded files, tracking of file history using a blockchain ledger, and real-time notifications for shared files. The system is particularly useful for organizations requiring safe internal file sharing or personal users seeking high data privacy.

Done by: John Doe, Jane Smith, and Alan Brown

Technologies Used: Blockchain (Ethereum), IPFS (InterPlanetary File System), Node.js, React.js
